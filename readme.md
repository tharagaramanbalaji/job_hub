# JobHub

A full-stack web application for a simple job listing platform. This project allows users to register as either a "Job Seeker" or a "Job Poster". Job posters can create new job listings, while job seekers can browse, search, filter, and save jobs they are interested in.

## Features

*   **User Authentication**: Secure registration and login for two distinct user roles (Seeker and Poster).
*   **Job Posting**: Job Posters can post new job opportunities.
*   **Job Browsing**: Users can view all available job listings.
*   **Search and Filter**: Dynamically search for jobs by keywords and filter by job type (Remote, Onsite, Hybrid).
*   **Save Jobs**: Job Seekers can save jobs to a personal list for later viewing.
*   **RESTful API**: A backend API built with Node.js and Express to handle data and user interactions.

## Technologies Used

*   **Backend**: Node.js, Express.js
*   **Frontend**: HTML, CSS, Vanilla JavaScript
*   **Database**: MySQL
*   **Dependencies**: `mysql2`, `bcrypt`, `express`

## Setup and Installation

To get this project running on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    cd jobhub
    ```

2.  **Install Dependencies:**
    Make sure you have [Node.js](https://nodejs.org/) installed. Then, run the following command in the project root to install the required packages:
    ```bash
    npm install
    ```

3.  **Database Setup:**
    *   Ensure you have a MySQL server running.
    *   Create a new database and user.
    *   Execute the SQL commands in the [`database_setup.sql`](c:\Users\Admin\Desktop\jobhub\database_setup.sql) file to create the necessary tables and insert sample data.

4.  **Configure Database Connection:**
    Open the [`server.js`](c:\Users\Admin\Desktop\jobhub\server.js) file and update the `mysql.createConnection` object with your MySQL host, user, password, and database name.

    ```javascript
    // filepath: c:\Users\Admin\Desktop\jobhub\server.js
    // ...existing code...
    const connection = mysql.createConnection({
        host: 'your_mysql_host',       // e.g., 'localhost'
        user: 'your_mysql_username',   // e.g., 'root'
        password: 'your_mysql_password',
        database: 'job_listings_db'
    });
    // ...existing code...
    ```

5.  **Start the Server:**
    Run the following command to start the application:
    ```bash
    npm start
    ```
    The server will be running at `http://localhost:3000`.

## Usage

Once the server is running, open your web browser and navigate to `http://localhost:3000`. You can:
*   Register as a 'Job Seeker' to search and save jobs.
*   Register as a 'Job Poster' to post new jobs.
*   Log in with your credentials to access role-specific features.
